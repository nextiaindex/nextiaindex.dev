# What is Codeberg?
Codeberg is another version of GitHub powered by Forgejo.

# Why I'm using Codeberg?

- Fork. I can flex my forked repository to make me look like an experienced programmer although I'm not
- Packages. This is the best way to host your binary files. Just upload them in Packages
